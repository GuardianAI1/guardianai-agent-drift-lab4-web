# LAB5 — Closure Boundary Matrix

## Objective
Map the boundary between closure, propagation, and constraint damping across gain/reanchor/doubt conditions.

## GuardianAI framing
GuardianAI does not react to single outputs. This lab maps when persistent closure remains revisable and when it approaches a consequence-bearing boundary.

## Core finding
Closure and amplification persistence can be separated by anti-gain controls.

## Focus metrics
- Closure yes/no and onset turn/cycle
- Amplification persistence turn/cycle
- Propagation yes/no
- Commitment streak max
