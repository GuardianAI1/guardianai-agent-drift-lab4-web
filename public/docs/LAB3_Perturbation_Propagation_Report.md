# LAB3 — Perturbation Propagation

## Objective
Measure how a controlled perturbation propagates in a recursive multi-agent loop once structural closure begins.

## GuardianAI framing
GuardianAI does not react to single outputs. In this lab it evaluates whether a trajectory remains closed over time rather than reopening under recursive reuse.

## Core finding
Errors amplify once closure forms.

## Focus metrics
- Amplification onset turn/cycle
- Amplification persistence turn/cycle
